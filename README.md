# Create Nuxt for KYG
> forked [nuxt/create-nuxt-app](https://github.com/nuxt/create-nuxt-app)

## Usage
### npm
```bash
$ npm install -g create-nuxt--kyg
$ create--nuxt--kyg <project-name>
```

### yarn
```bash
$ yarn create nuxt--kyg <project-name>
```
